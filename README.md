# SINUX_F1
sinux f1 is a open hardware evalution board for F1c100s chip


### supported peripherals : 
* I2C
* SPI
* SD Card
* UART
* SPI Flash
* lradc
* LCD (40pin fpc)
* USB
* Backlight

### Other features
* Resistive touch (NS2009 chip)
* Capacitive touch connector
* USB Hub (FE1.1S chip)
* Wifi (RIL8188EUS)
